
  # A/B Testing Project

  This demo project is focused on A/B test in landing pages, allowing users to create different variations of an app and see the difference in behaviour of the user, the project is a simplified version of a possible complete platform with authentication, a complete backend and more functionalities.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
